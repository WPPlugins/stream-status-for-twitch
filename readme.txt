Add a customizable, stylishly animated button to your WordPress site and let your visitors know when you're livestreaming.

== Description ==

Stream Status for Twitch is a highly noticeable yet extremely unobtrusive way to let your visitors know when you're livestreaming on Twitch. Beautifully animated and extensively customizable.

**Features:**

* Quick and simple setup
* Optionally show only when stream is live
* Add your own 'live now' and 'currently offline' texts
* Positioning and customization options included
* Setup and customization can be done quickly and in real-time via the WordPress Live Customizer

== Installation ==

1. Upload /stream-status-for-twitch/ to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the "Appearance->Customize->Stream Status for Twitch Plugin" screen to configure and customize the plugin
